# 「Discord 通話呼び出しBOT」
# 概要：Discordサーバー内の特定ユーザーに対して、ボイスチャンネルへの招待リンクをDMで送信し、通話へ参加するまで一定間隔で通知を続けるBOTです。ユーザーが通話へ参加したことを検知すると、通知を停止します。
# 使用技術：
          Python
          discord.py
          asyncio
          Discord Bot API
# 工夫した点：
          ボイスチャンネルの招待リンクを自動生成
          DM送信に失敗した場合の例外処理を実装
          ユーザーの通話参加を監視し、参加後は自動的に通知を終了
          通知回数を記録して表示
# Discord 通話呼び出しBOT
          Discord上で通話への参加を促すために開発したBOTです。指定したユーザーへボイスチャンネルの招待リンクを定期的に送信し、参加状況を監視します。非同期処理を利用して通知と監視を同時に実行できるよう設計しました。例外処理やDiscord APIの利用方法について学ぶことができました。
